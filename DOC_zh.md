# IK配置
在我们的ik配置中，如`smplx_to_g1.json`，您可能会找到以下参数。我在这里添加注释以便您理解。
```json
"ik_match_table1": {
        "pelvis": [ # 机器人的身体名称
            "pelvis", # 对应的人体身体名称，这里我们以"pelvis"为例
            100, # 跟踪3D位置(xyz)的权重
            10, # 跟踪3D旋转的权重
            [
                0.0, # 添加到人体"pelvis" x的偏移量
                0.0, # 添加到人体"pelvis" y的偏移量
                0.0 # 添加到人体"pelvis" z的偏移量
            ],
            [
                # 应用于人体"pelvis"的旋转(表示为四元数)。顺序遵循标量优先(wxyz)
                0.5,
                -0.5,
                -0.5,
                -0.5
            ]
        ],
      ...
```
